Primeiros Passos com a API

1. Crie uma Chave de API: Vá até a aba Chaves de API, informe seu nome e crie uma chave de acesso (WRITE ou ADMIN para poder cadastrar dados).

2. Defina a Chave Ativa: Copie a chave gerada e clique em "Usar esta Chave" para que o painel envie ela automaticamente no cabeçalho X-API-Key de todas as chamadas.

3. Cadastre Entidades Relacionadas: Crie primeiro as Ligas e Estádios antes de criar Times, pois eles precisam dessas associações. Em seguida, cadastre Jogadores associados a times e agende Partidas.

4. Swagger API Docs: Acesse a documentação Swagger interativa gerada pelo back-end em /swagger-ui.html para testar os endpoints diretamente.
